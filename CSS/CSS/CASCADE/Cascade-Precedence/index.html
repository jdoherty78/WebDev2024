Cascading Style Sheet:
    hierarchichal order
    only relevant when conflicting css rules
    which rules get applied, and which get ignored
  1. External stylesheet (site specific)
  2. Internal - via style block (page specific)
  3. Inline - via style tag (very specific)

  1. Position
    • the one in the lowest position
    • li {color: blue;
      color: red;} <-- red will "win"
  2. Specificity
    • the selector with the highest specificity will value will "win"
    • ie. From General to Most Specific:
      1. ELEMENT: li {color: blue;}
      2. CLASS .first-class {color: red;}
      3. ATTRIBUTE: li [draggable]{color:purple;}
      4. ID: #first-id {color: orange;}
      5. OR... use the keyword color: green !important; <-- will take precedence

  4. Importance
